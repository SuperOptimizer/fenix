# Design notes

Per-subsystem design documents (deeper than a module's `CLAUDE.md`, narrower than an ADR).
Add `NN-<subsystem>.md` as designs solidify — e.g. the `.fxvol` container byte layout, the
wavelet bitplane scan order, the diffeomorphic-fit energy + OOC schedule, the surface
container schema. Until then, the authoritative design lives in each module's `CLAUDE.md`,
the [ADRs](../adr/), and the synthesized predecessor [research](../research/).
